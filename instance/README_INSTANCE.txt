Ce dossier contient les données persistantes de l'application.
- patients.json : liste des patients au format JSON.
- patients.sqlite3 : base SQLite facultative (table `patients`).
- first_run.marker : fichier marqueur de premier lancement.

Vous pouvez remplacer patients.json par vos propres données en
respectant le schéma [{"id":"...","displayName":"..."}].
